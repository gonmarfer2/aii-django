Se pide (1 punto) CONSTRUIR UNA WEB CON UN MENÚ PRINCIPAL que permita las siguientes opciones:

  • (2.5 puntos) LA BASE DE DATOS. Muestre un formulario de confirmación. Si se acepta, borrar la base de datos y volverla a crear con los datos del dataset. Después MOSTRAR INFORMACIÓN DE LOS REGISTROS ALMACENADOS EN CADA TABLA que hayáis definido en el modelo.
  • (1 punto) CARGAR RECSYS . Que cargue TODOS LOS DATOS NECESARIOS para el Sistema de Recomendación y otros apartados.
  • (1.5 puntos) ANIMES POR GÉNERO. Muestre un formulario con un ChoiceField con la lista de géneros que hay en la BD. Cuando se seleccione un género muestra el total de animes de ese género agrupados por formato de emisión (Título, Géneros, Formato de emisión y Número de episodios).
  • (1.5 puntos) MEJORES ANIMES. Muestre los tres animes con una puntuación media más alta (Título y puntuación media).
  • (1.5 puntos) RECOMENDAR ANIMES. Muestre un formulario para introducir un usuario (IdUsuario), le recomiende tres animes que no haya puntuado (Título y puntuación propuesta por el sistema de recomendación), usando un sistema de recomendación Basado en ítems.
